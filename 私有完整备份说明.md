# 私有完整备份说明

本项目的源码仓库只保存可公开给维护人员阅读的源码、部署脚本和文档。生产密钥、数据库、Caddy 证书、Redis 数据、NVIDIA 账号池和备份包都不应该进入 Git。

## 服务器生产备份

在服务器上执行：

```bash
cd /opt/sub2api-nvidia/cloud-deploy
./scripts/backup.sh
ls -lh backups | tail -20
```

备份文件位于服务器：

```text
/opt/sub2api-nvidia/cloud-deploy/backups/sub2api-postgres-*.sql.gz
/opt/sub2api-nvidia/cloud-deploy/backups/sub2api-files-*.tar.gz
```

这些备份包含敏感内容：

- `cloud-deploy/.env`
- `cloud-deploy/secrets/`
- adapter SQLite 数据
- Caddy 证书和私钥
- Redis 数据
- Sub2API 本地数据目录

请只保存在你控制的私有磁盘、加密网盘或服务器本地，不要上传到 GitHub、公开网盘、聊天记录或论坛。

## 本机空间不足时的建议

你的本机 C 盘空间较紧张时，不建议把完整生产备份下载到本机。优先让备份留在服务器本地；需要长期保存时，复制到外接硬盘或空间充足的私有存储。
