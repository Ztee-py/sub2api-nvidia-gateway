# 私有完整备份说明

本文件用于说明 `release/sub2api-nvidia-gateway-private-full-backup.zip`。

这个私有备份包用于个人留档，包含敏感内容，可能包括：

- 根目录 `.env`
- `cloud-deploy/.env`
- 本地 SQLite 数据库 `sub2api.db`
- 本地日志文件
- Python 缓存目录
- 脱敏源码包副本

不要把这个私有备份包上传到聊天、网盘公开链接、GitHub、论坛或发给其他人。

如果另一个聊天需要继续开发、优化或排查问题，请上传脱敏包：

```text
release/sub2api-nvidia-gateway-source.zip
```

不要上传私有包：

```text
release/sub2api-nvidia-gateway-private-full-backup.zip
```

生产服务器上的 PostgreSQL 数据、Caddy 证书、Redis 数据和服务器本地备份位于云服务器 `/opt/sub2api-nvidia/cloud-deploy/`，不一定完整存在于本机源码目录。服务器上的生产备份请优先留在服务器本地：

```bash
cd /opt/sub2api-nvidia/cloud-deploy
./scripts/backup.sh
ls -lh backups | tail -20
```
